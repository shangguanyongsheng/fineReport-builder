# 样式模板配置

从 FinanceCreditContractAnalysis.cpt 提取的样式模板，可直接在报表生成时选择使用。

---

## 样式模板列表

### 模板 1: 经典蓝（默认）

```
适用场景：标准报表，蓝色表头 + 边框
```

| 索引 | 名称 | 对齐 | 边框 | 背景 | 用途 |
|------|------|------|------|------|------|
| 0 | 表头左列 | 左 | 有 | 蓝色 | 第一列表头（固定列） |
| 1 | 表头 | 左 | 有 | 蓝色 | 普通表头 |
| 2 | 数据 | 左 | 有 | 无 | 普通数据单元格 |

| 4 | 默认 | 中 | 无 | 无 | 默认样式 |

```json

  {"name": "表头左列", "horizontal_alignment": "2", "font": {"name": "SimSun", "size": "80"}, "background": "-1447425", "border": true},
  {"name": "表头", "horizontal_alignment": "2", "font": {"name": "宋体", "size": "80"}, "background": "-1447425", "border": true},
  {"name": "数据", "horizontal_alignment": "2", "font": {"name": "SimSun", "size": "80"}, "border": true},
```

## 数字格式

| 格式 | 说明 | 示例 |
|------|------|------|
| #,##0.00 | 千分位 + 两位小数 | 1,234.56 |
| #,##0 | 千分位 + 整数 | 1,234 |
| 0.00% | 百分比 | 12.34% |
| ¥#,##0.00 | 人民币 | ¥1,234.56 |

## 样式索引使用

在单元格配置中指定 `style_index`：

```json
{
  "column": 0,
  "row": 0,
  "value": "金额",
  "style_index": 1    // 使用第 2 个样式（表头）
}
```

**常用映射：**
- `style_index: 0` → 表头左列/标题
- `style_index: 1` → 表头
- `style_index: 2` → 普通数据
- `style_index: 3` → 金额/数值
