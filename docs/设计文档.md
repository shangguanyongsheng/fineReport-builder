# FineReport Builder 设计文档

## 1. 项目概述

FineReport Builder 是一个帆软报表自动化构建工具，支持多种输入方式生成 .cpt 报表文件。

### 1.1 核心目标

- **降低门槛**：无需掌握帆软设计器，通过自然语言/Excel 即可生成报表
- **提高效率**：自动化生成报表骨架，减少重复性工作
- **增强协作**：Web 服务支持团队共享使用

### 1.2 技术栈

| 层级 | 技术 |
|------|------|
| Web 服务 | Flask + Flask-CORS |
| 前端 | Bootstrap 5 + Bootstrap Icons |
| 文件解析 | openpyxl (Excel), xml.etree (CPT) |
| 部署 | Python 3.8+, 支持 Gunicorn |

### 2.2 模块说明

| 模块 | 职责 | 文件 |
|------|------|------|
| **Web 服务** | 提供 HTTP API 和页面 | `web/app.py` |
| **Agent** | 协调各模块工作 | `agent/__init__.py` |
| **需求解析** | 解析自然语言需求 | `agent/requirement_parser.py` |

## 3. 核心流程

### 3.1 从自然语言生成报表

```
┌─────────────────────────────────────────────────────────────┐
│  用户输入: "创建销售报表，按区域分组，显示金额、数量"          │
└───────────────────────────┬─────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  RequirementParser.parse()                                  │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 1. 提取标题 → "销售报表"                              │   │
│  │ 2. 提取数据源 → table=sales, fields=[金额,数量]      │   │
│  │ 3. 提取筛选条件 → 日期范围、区域                      │   │
│  │ 4. 提取分组 → group_by=[区域]                        │   │
│  └─────────────────────────────────────────────────────┘   │
└───────────────────────────┬─────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  ReportSpec (结构化配置)                                    │
│  {                                                          │
│    "title": "销售报表",                                     │
│    "data_source": {                                         │
│      "table": "sales",                                      │
│      "fields": ["金额", "数量"],                            │
│      "group_by": ["区域"]                                   │
│    },                                                       │
│    "filter_controls": [                                     │
│      {"name": "startDate", "type": "DateEditor"},           │
│      {"name": "region", "type": "TreeComboBoxEditor"}       │
│    ]                                                        │
│  }                                                          │
└───────────────────────────┬─────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  CPTGenerator.generate()                                    │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ 1. 生成 TableDataMap (数据源定义)                    │   │
│  │ 2. 生成 ReportParameterAttr (参数面板)               │   │
│  │ 3. 生成 CellElementList (单元格列表)                 │   │
│  │ 4. 生成 StyleList (样式列表)                         │   │
│  └─────────────────────────────────────────────────────┘   │
└───────────────────────────┬─────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  输出: sales_report.cpt (帆软报表文件)                       │
└─────────────────────────────────────────────────────────────┘
```

## 4. .cpt 文件结构

### 4.1 CPT 文件架构

```xml
<?xml version="1.0" encoding="UTF-8"?>
<WorkBook xmlVersion="20211223" releaseVersion="11.5.0">
    
    <!-- ========== 1. 数据源映射 ========== -->
    <TableDataMap>
        <TableData name="sales_data" class="com.fr.data.impl.DBTableData">
            <Connection class="com.fr.data.impl.NameDatabaseConnection">
                <DatabaseName><![CDATA[sales_db]]></DatabaseName>
            </Connection>
            <Query><![CDATA[SELECT * FROM sales WHERE date >= '${startDate}']]></Query>
            <Parameters>
                <Parameter>
                    <Attributes name="startDate"/>
                    <O><![CDATA[]]></O>
                </Parameter>
            </Parameters>
        </TableData>
        
        <!-- ClassTableData 示例 -->
        <TableData name="credit_data" class="com.fr.data.impl.ClassTableData">
            <Parameters>
                <Parameter><Attributes name="orgId"/></Parameter>
            </Parameters>
            <ClassTableDataAttr className="com.yocyl.fr.engine.tableData.CreditData"/>
        </TableData>
    </TableDataMap>
    
    <!-- ========== 2. Web 属性 ========== -->
    <ReportWebAttr>
        <Title><![CDATA[销售报表]]></Title>
    </ReportWebAttr>
    
    <!-- ========== 3. 参数面板 ========== -->
    <ReportParameterAttr>
        <Attributes showWindow="true"/>
        <ParameterUI>
            <Layout>
                <!-- 日期控件 -->
                <Widget>
                    <InnerWidget class="com.fr.form.ui.DateEditor">
                        <WidgetName name="startDate"/>
                        <LabelName name="开始日期"/>
                    </InnerWidget>
                    <BoundsAttr x="100" y="10" width="135" height="28"/>
                </Widget>
                
                <!-- 下拉树控件 -->
                <Widget>
                    <InnerWidget class="com.fr.form.ui.TreeComboBoxEditor">
                        <WidgetName name="orgId"/>
                        <LabelName name="组织"/>
                    </InnerWidget>
                    <BoundsAttr x="250" y="10" width="135" height="28"/>
                </Widget>
                
                <!-- 查询按钮 -->
                <Widget>
                    <InnerWidget class="com.fr.form.ui.FormSubmitButton">
                        <WidgetName name="Search"/>
                        <Text><![CDATA[查询]]></Text>
                    </InnerWidget>
                </Widget>
            </Layout>
        </ParameterUI>
    </ReportParameterAttr>
    
    <!-- ========== 4. 报表主体 ========== -->
    <Report class="com.fr.report.worksheet.WorkSheet" name="Sheet1">
        <CellElementList>
            <!-- 表头 -->
            <C c="0" r="0">
                <O><![CDATA[区域]]></O>
                <s>0</s>
            </C>
            
            <!-- 数据绑定单元格 -->
            <C c="0" r="1">
                <O t="DSColumn">
                    <Attributes dsName="sales_data" columnName="region"/>
                </O>
                <Expand dir="0"/>  <!-- 纵向扩展 -->
            </C>
            
            <!-- 公式单元格 -->
            <C c="2" r="10">
                <O t="XMLable" class="com.fr.base.Formula">
                    <Attributes><![CDATA[=SUM(C2)]]></Attributes>
                </O>
            </C>
        </CellElementList>
    </Report>
    
    <!-- ========== 5. 样式列表 ========== -->
    <StyleList>
        <Style style_name="默认">
            <FRFont name="simhei" style="0" size="72"/>
            <Background name="NullBackground"/>
        </Style>
    </StyleList>
    
</WorkBook>
```

## 6. 数据结构

### 6.1 核心数据类

```python
# 数据源规格
@dataclass
class DataSourceSpec:
    name: str           # 数据源名称
    database: str       # 数据库连接
    table: str          # 表名
    fields: List[str]   # 字段列表
    filters: List[str]  # 筛选条件
    group_by: List[str] # 分组字段
    aggregations: Dict  # 聚合函数

# 筛选控件规格
@dataclass
class FilterControlSpec:
    name: str           # 控件名称
    label: str          # 显示标签
    control_type: str   # 控件类型
    data_type: str      # 数据类型
    options: Dict       # 下拉选项

# 单元格规格
@dataclass
class CellElement:
    column: int         # 列号
    row: int            # 行号
    value: str          # 值
    value_type: str     # 值类型
    data_source: str    # 数据源
    column_name: str    # 列名
    expand_dir: str     # 扩展方向
```

## 8. 部署指南

### 8.1 开发环境

```bash
# 克隆项目
git clone https://github.com/shangguanyongsheng/fineReport-builder.git
cd fineReport-builder

# 安装依赖
pip install -r requirements.txt

# 启动 Web 服务
python run.py web --port 5000

# 启动 GUI（可选）
python run.py gui
```

### 8.2 生产环境

```bash
# 使用 Gunicorn
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 web.app:app

# 使用 systemd 服务
sudo cat > /etc/systemd/system/fineReport-builder.service << EOF
[Unit]
Description=FineReport Builder Web Service
After=network.target

[Service]
User=admin
WorkingDirectory=/home/admin/.openclaw/workspace/projects/fineReport-builder
ExecStart=/usr/bin/gunicorn -w 4 -b 0.0.0.0:5000 web.app:app
Restart=always

[Install]
WantedBy=multi-user.target
EOF

sudo systemctl enable fineReport-builder
sudo systemctl start fineReport-builder
```
