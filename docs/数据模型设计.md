# fineReport-builder V2 数据模型设计

## 1. 数据库数据源配置

```json
{
  "name": "sales_data",
  "type": "database",
  "database": "cfs-report",
  "sql": "SELECT region, product, amount, quantity FROM sales WHERE date >= '${startDate}' AND date <= '${endDate}'",
  "column_mapping": {
    "A": "region",
    "B": "product",
    "C": "amount",
    "D": "quantity"
  }
}
```

**字段说明：**
- `name`: 数据源名称
- `type`: 固定为 "database"
- `database`: 数据库连接名称
- `sql`: SQL 查询语句，支持 `${参数名}` 占位符
- `column_mapping`: Excel 列到数据库字段的映射（不支持一对多）

## 2. Class 数据源配置

```json
{
  "name": "credit_data",
  "type": "class",
  "class_name": "com.yocyl.fr.engine.tableData.finance.CreditContractDetailData",
  "return_fields": [
    {"name": "contractNo", "type": "string", "description": "合同编号"},
    {"name": "amount", "type": "number", "description": "金额"},
    {"name": "orgName", "type": "string", "description": "组织名称"}
  ],
  "parameter_template": {
    "orgId": "",
    "startDate": "2026-03-02",
    "endDate": "",
    "indexInfo": [],
    "dimensions": []
  }
}
```

**字段说明：**
- `name`: 数据源名称
- `type`: 固定为 "class"
- `class_name`: Java 类全路径
- `return_fields`: 返回字段列表
- `parameter_template`: 入参模板
  - 空值 `""` → 从筛选组件获取
  - 有值 `"2026-03-02"` → 写死的默认值
  - 空数组 `[]` → 从筛选组件获取（数组类型）

## 3. 筛选组件配置

```json
{
  "components": [
    {
      "label": "开始日期",
      "code": "startDate",
      "type": "DateEditor",
      "default_value": "",
      "placeholder": "",
      "options": {}
    },
    {
      "label": "结束日期",
      "code": "endDate", 
      "type": "DateEditor",
      "default_value": "",
      "placeholder": "",
      "options": {}
    },
    {
      "label": "组织机构",
      "code": "orgId",
      "type": "TreeComboBoxEditor",
      "default_value": "",
      "placeholder": "请选择组织",
      "options": {}
    },
    {
      "label": "合同编号",
      "code": "contractNo",
      "type": "TextEditor",
      "default_value": "",
      "placeholder": "请输入合同号模糊查询",
      "options": {}
    },
    {
      "label": "区域",
      "code": "region",
      "type": "ComboBox",
      "default_value": "",
      "placeholder": "请选择",
      "options": {
        "east": "华东",
        "south": "华南",
        "north": "华北",
        "west": "西部"
      }
    }
  ],
  "layout": {
    "components_per_row": 5,
    "auto_arrange": true
  }
}
```

**字段说明：**
- `label`: 文本组件（说明），用户看到的标签
- `code`: 参数名，用于传值给数据集
- `type`: 控件类型
  - `TextEditor`: 文本输入框
  - `DateEditor`: 日期选择器
  - `ComboBox`: 下拉选择框
  - `TreeComboBoxEditor`: 树形下拉
  - `ComboCheckBox`: 多选下拉
  - `NumberEditor`: 数字输入框
- `default_value`: 默认值
- `placeholder`: 占位提示文字
- `options`: 下拉选项（仅 ComboBox 类型）

## 5. 参数绑定流程

```
┌─────────────────────────────────────────────────────────────────┐
│                    参数绑定流程                                  │
└─────────────────────────────────────────────────────────────────┘

1. 用户配置筛选组件
   ┌─────────────────────────────────┐
   │ 开始日期: startDate             │
   │ 组织机构: orgId                 │
   │ 合同编号: contractNo            │
   └─────────────────────────────────┘
              │
              ▼
2. 系统自动检测数据源入参
   ┌─────────────────────────────────┐
   │ 数据库数据集 SQL:               │
   │   ...WHERE date >= '${startDate}'│
   │                                 │
   │ Class 数据集入参:               │
   │   {"orgId": "", "startDate": ""}│
   └─────────────────────────────────┘
              │
              ▼
3. 自动匹配绑定
   ┌─────────────────────────────────┐
   │ startDate → 筛选组件 startDate  │
   │ orgId → 筛选组件 orgId          │
   │                                 │
   │ ⚠️ 警告: indexInfo 无对应组件   │
   └─────────────────────────────────┘
              │
              ▼
4. 生成 .cpt 文件
   - 参数面板组件自动排列（一行 5 对）
   - 数据源参数自动绑定
```
