# 改动区域抽象设计

## 核心思路

**不改代码，只替换 JSON 配置**

```
模板文件 (.cpt) + 改动配置 (JSON) = 新报表
```

---

## 改动配置 JSON 结构

```json
{
  "template": "明细",  // 模板类型
  
  "filter_area": {
    "operation": "replace",  // replace | modify
    "components": [
      {"label": "开始日期", "code": "startDate", "type": "DateEditor"},
      {"label": "结束日期", "code": "endDate", "type": "DateEditor"},
      {"label": "组织机构", "code": "orgId", "type": "TreeComboBoxEditor"}
    ],
    "layout": {
      "max_per_row": 5,
      "label_width": 70,
      "input_width": 135,
      "gap": 10,
      "row_gap": 40,
      "start_x": 10,
      "start_y": 10
    }
  },
  
  "data_area": {
    "operation": "replace",  // replace | modify
    "header_row": 0,
    "data_row": 1,
    "columns": [
      {"name": "合同编号", "field": "contractNo", "style": 2},
      {"name": "合同名称", "field": "contractName", "style": 2},
      {"name": "金额", "field": "amount", "style": 3},
      {"name": "状态", "field": "status", "style": 2}
    ]
  },
  
  "data_source": {
    "operation": "modify",  // replace | modify
    "name": "CreditContractDetailData",
    "parameters": {
      "orgId": "",
      "startDate": "",
      "endDate": ""
    }
  }
}
```

---

## 操作类型说明

### filter_area

| 操作 | 说明 |
|------|------|
| `replace` | 清空现有组件，按配置重新生成 |
| `modify` | 保留现有组件，增删改 |

### data_area

| 操作 | 说明 |
|------|------|
| `replace` | 清空现有数据列，按配置重新生成 |
| `modify` | 保留现有列，增删改 |

### data_source

| 操作 | 说明 |
|------|------|
| `replace` | 替换整个数据源 |
| `modify` | 只修改参数 |

## 位置计算规则

```
一行 max_per_row 对组件

组件 i 的位置：
  row = i // max_per_row
  col = i % max_per_row
  
  label_x = start_x + col * (label_width + input_width + gap)
  input_x = label_x + label_width + gap
  y = start_y + row * row_gap
```

**示例（max_per_row=5, start_x=10, start_y=10）：**

```
第1行 (row=0):
  组件0: label_x=10, input_x=90, y=10
  组件1: label_x=225, input_x=305, y=10
  组件2: label_x=440, input_x=520, y=10
  组件3: label_x=655, input_x=735, y=10
  组件4: label_x=870, input_x=950, y=10

第2行 (row=1):
  组件5: label_x=10, input_x=90, y=50
  ...
```

## 数据列处理规则

### replace 模式

```
1. 删除所有现有数据列单元格（row >= header_row）
2. 按 columns 配置重新生成
3. 表头行：设置 name
4. 数据行：设置 field（DSColumn 绑定）
```

### modify 模式

```
1. 对比现有列和目标列
2. 删除多余列
3. 添加新列（复制最后一列）
4. 更新已有列
```

## 实现要点

### 1. 筛选组件生成

```python
def generate_filter_components(config):
    """按配置生成筛选组件 XML"""
    
    components = config['filter_area']['components']
    layout = config['filter_area']['layout']
    
    xml_nodes = []
    
    for i, comp in enumerate(components):
        row = i // layout['max_per_row']
        col = i % layout['max_per_row']
        
        label_x = layout['start_x'] + col * (layout['label_width'] + layout['input_width'] + layout['gap'])
        input_x = label_x + layout['label_width'] + layout['gap']
        y = layout['start_y'] + row * layout['row_gap']
        
        # 生成 Label XML
        label_xml = f'''
        <Widget class="com.fr.form.ui.container.WAbsoluteLayout$BoundsWidget">
          <InnerWidget class="com.fr.form.ui.Label">
            <WidgetName name="label_{comp['code']}"/>
            <widgetValue><O>{comp['label']}</O></widgetValue>
          </InnerWidget>
          <BoundsAttr x="{label_x}" y="{y}" width="{layout['label_width']}" height="28"/>
        </Widget>
        '''
        
        # 生成输入控件 XML
        input_xml = f'''
        <Widget class="com.fr.form.ui.container.WAbsoluteLayout$BoundsWidget">
          <InnerWidget class="com.fr.form.ui.{comp['type']}">
            <WidgetName name="{comp['code']}"/>
          </InnerWidget>
          <BoundsAttr x="{input_x}" y="{y}" width="{layout['input_width']}" height="28"/>
        </Widget>
        '''
        
        xml_nodes.append(label_xml)
        xml_nodes.append(input_xml)
    
    return xml_nodes
```

### 2. 数据列生成

```python
def generate_data_columns(config):
    """按配置生成数据列 XML"""
    
    columns = config['data_area']['columns']
    header_row = config['data_area']['header_row']
    data_row = config['data_area']['data_row']
    
    xml_nodes = []
    
    for i, col in enumerate(columns):
        # 表头单元格
        header_xml = f'''
        <C c="{i}" r="{header_row}" s="{col.get('style', 0)}">
          <O t="Text"><![CDATA[{col['name']}]]></O>
        </C>
        '''
        
        # 数据单元格
        data_xml = f'''
        <C c="{i}" r="{data_row}" s="{col.get('style', 0)}">
          <O t="DSColumn">
            <Attributes dsName="{config['data_source']['name']}" columnName="{col['field']}"/>
          </O>
        </C>
        '''
        
        xml_nodes.append(header_xml)
        xml_nodes.append(data_xml)
    
    return xml_nodes
```
